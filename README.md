
#  FridgeChef

FridgeChef is a smart recipe suggestion web app that helps you **decide what to cook** using the ingredients you already have in your fridge.  
Simply input your available ingredients, and FridgeChef will generate delicious recipe ideas instantly!

---

##  Features

-  **Smart Ingredient Input:** Enter multiple ingredients that you currently have.  
-  **Recipe Recommendations:** AI-powered or API-based recipe suggestions using those ingredients.  
-  **Nutritional Info:** View estimated calories and nutrition details for each recipe.  
-  **Save Recipes:** Save your favorite recipes for later viewing.  
-  **Clean & Modern UI:** Minimalistic design with light/dark mode support.  
-  **Search & Filter:** Search recipes by name, ingredient, or category.
